# Scale Circle 🎵

五度圏ベースのスケール練習ツール。ブラウザだけで動くシングルページアプリです。

**→ https://scale-practice-delta.vercel.app**

## 機能

- **五度圏ホイール** — ドラッグ／スワイプで回転させてルートキーを選択
- **20種類のスケール** — 教会旋法・ペンタトニック・ジャズスケール・ハーモニックマイナーなど
- **五線譜表示** — 選択したスケールをト音記号つきで表示（加線対応）
- **ピアノ音源再生** — Salamander Grandピアノサンプルによる本物のピアノ音（シンセフォールバックあり）
- **BPM／オクターブ調整** — スケールを好みのテンポで昇降演奏
- **ライト／ダークテーマ** — ヘッダーのボタンで切り替え

## スケール一覧

| カテゴリ | スケール |
|---|---|
| 教会旋法 | アイオニアン、ドリアン、フリジアン、リディアン、ミクソリディアン、エオリアン、ロクリアン |
| ペンタトニック | メジャーペンタ、マイナーペンタ |
| ジャズ | ブルーノート、ドミナント7thペンタ、メジャー／マイナービバップ、ドリアンビバップ、リディアンドミナント、オルタード、ホールトーン、ディミニッシュ(HW/WH) |
| その他 | ハーモニックマイナー、メロディックマイナー、ハーモニックメジャー |

## 使い方

ホイールをドラッグしてルートキーを選び、スケールボタンをタップ → ▶ 再生。

ローカルで開く場合はファイルをそのままブラウザにドラッグするだけ（ビルド不要）。

## 技術スタック

Vanilla HTML / CSS / JavaScript（フレームワークなし）  
音源: [Salamander Grand Piano](https://github.com/urish/cinto) via Tone.js CDN
